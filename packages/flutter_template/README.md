## フォルダ構成
新しいパッケージを追加する場合を考慮したフォルダ構成を作成。
参考: https://dash-overflow.net/articles/getting_started/#:~:text=step%201%3A%20(optional)%20use%20the%20%2Fpackages%20folder%20convention
```
.git/
.gitignore
packages/
  my_project/
    pubspec.yaml
    lib/
    ...
  my_first_package/
    pubspec.yaml
    ...
```
## library

|ファイル|説明|補足|
